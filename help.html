Generic FAQ

Downloads get stuck, what to do?

Make sure to diagnose the issue. Currently, you can start the app from the terminal to debug.

Deleted files aren’t removed from my PC.

This is expected behavior for now. We’ll be fixing this in a future update.

How to report an issue/feature request?

Visit our main project page and create a new issue: GitHub Issues.

Streamrip FAQ

Streamrip downloads (Qobuz, Deezer, Tidal) get stuck, what to do?

Try running $custom_rip config reset. If unresolved, try downloading from your terminal. If it fixes, create a new issue
